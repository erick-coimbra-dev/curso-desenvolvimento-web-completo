# 💻 Curso: Desenvolvimento Web Completo (20 Cursos + 20 Projetos)

Repositório dedicado para centralizar a evolução dos meus estudos, códigos teóricos, laboratórios e os projetos práticos construídos ao longo do treinamento da **Udemy**.

---

### 🎯 Objetivo do Projeto
Complementar a minha formação na faculdade de Análise e Desenvolvimento de Sistemas (ADS), construindo uma base sólida de engenharia de software e arquitetura de aplicações através de projetos práticos e reais.

---

### 🧰 Tecnologias Estudadas & Praticadas
O curso aborda um ecossistema completo de desenvolvimento, onde estou praticando e aplicando:

- **Front-End:** HTML5, CSS3, SASS, Bootstrap 5, JavaScript (ES6) e JQuery.
- **Back-End & Lógica:** PHP e Orientação a Objetos.
- **Banco de Dados:** MySQL.
- **Ferramentas:** Controle de versão com Git e GitHub, Padrão de Arquitetura MVC e APIs.

---

### 📁 Organização dos Arquivos
Os códigos estão estruturados de forma cronológica para facilitar a consulta rápida:
- `/Modulos`: Exercícios conceituais de sintaxe e lógica separados por tecnologia.
- `/Projetos`: Os sistemas e clones de plataformas reais desenvolvidos do zero.
